Cloud integration

“Bridges” are created using nChannel Connectors. Registered customers download connectors for each system they want to connect to the nChannel platform. Connectors act as “listeners” that listen for any changes – such as new orders, or new items – made to any of your systems and automatically syncs changes to the nChannel portal.

Not only does the platform listen for changes, but it is smart enough to know what to do with changes it picks up. Customers can tell the system what to do and how to manage orders, items, and customer information. For example – when nChannel receives a web order, push it to the POS system – or if it receives an order for a particular item, push it to a selected supplier for fulfillment. You can tell the system to automatically download sales to your accounting system – or create a PO in the accounting system, when a work order is created in the POS system.
